AAD administration interface

See the administration manual for a more detailed description.

Edit existing data

Edit an experiment's details

Edit the type, facility, position of the animal, distance to sound source, test environment, medium, method, calibration, threshold, staircase procedure, method of constants and sedation state of an experiment.

Edit an animal's details

Edit the species, sex, liberty status, life stage, age or duration in captivity of an animal.

Edit an audiogram's data points

For each data point in an audiogram, edit the SPL, frequency, duration and reference values, remove or add new data points.

Delete an audiogram from the database

Delete an audiogram from the database. If the deleted audiogram is the only audiogram for the corresponding publication, facility, experiment or animal, these will be deleted too. Deletions are permanent, deleted audiograms cannot be undeleted, this action cannot be undone.

Add new data

Adding a new audiogram to the database requires 3 actions:
  1. Add the publication details
  2. Add the species details
  3. Create an experiment and add metadata about publication, method, etc.
  4. Finally, open the newly created a experiment and add the audiogram's data points

Add a publication

Add bibliographical details of a publication (Autors, Title, Journal, DOI, etc.). If the publication is already in the database, skip this step. Publication details can be retrieved automatically by DOI.

Add species details

Add taxonomic details of a new species (Latin name, vernacular name, etc.). If the species is already in the database, skip this step. Taxonomic details can be retrieved automatically from the Tree of Life database.

Add a new experiment to the database.

Create an experiment and add metadata about publication, method, etc.
